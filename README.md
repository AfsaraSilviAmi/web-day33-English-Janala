# ENGLISH JANALA <img width="25px" src="./assets/logo.png" />

English Janala is a vocabulary learning web app that helps users explore English words by levels, view word details, and practice pronunciation.

---

## ⚡ Features

- Display lessons/levels dynamically from API
- Show vocabulary cards with word, meaning, and pronunciation
- Highlight active lesson buttons
- View detailed word information in a modal:
  - Example sentences
  - Synonyms
- Search words dynamically
- Save favorite words in a separate section
- Play pronunciation using the browser’s SpeechSynthesis API
- Responsive and user-friendly UI
- Loading spinner while fetching data from API

---

## 🔗 API Endpoints

- **Get All Levels:**  
`https://openapi.programming-hero.com/api/levels/all`

- **Get Words by Level:**  
`https://openapi.programming-hero.com/api/level/{id}`

- **Get Word Details:**  
`https://openapi.programming-hero.com/api/word/{id}`

- **Get All Words:**  
`https://openapi.programming-hero.com/api/words/all`
